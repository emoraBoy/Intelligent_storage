##智能仓储管理系统
##项目描述：
    该项目是自己在参观学校智能制衣工厂，有感而发，正好当时正在学习Spring Boot,就选择做一个小demo当作练习。主要实现了客户管理、仓库管理、产品管理、库存管理。
